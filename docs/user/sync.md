# 同步与远程控制

CreatorWeave 支持远程控制功能，让你可以用手机或其他设备控制电脑上的 CreatorWeave。

## 远程控制设置

### 启用远程控制

1. 打开设置面板
2. 找到"远程控制"选项
3. 点击启用

### 生成连接码

启用后会显示：

- **连接地址**: 用于远程访问的 URL
- **连接二维码**: 扫码快速连接

### 安全设置

| 设置项 | 说明 |
|-------|------|
| 访问密码 | 设置密码保护连接 |
| 自动断开 | 空闲一定时间后自动断开 |

## 扫码连接

### 手机扫码

1. 打开 CreatorWeave 移动端（如果有）
2. 扫描电脑上的二维码
3. 输入密码（如设置了）

### 连接成功

连接成功后：

- 手机显示电脑上的 CreatorWeave 界面
- 可以进行对话、查看文件等操作
- 所有操作实时同步到电脑

## 文件变更同步

### 查看变更

在同步面板中查看：

- **新增文件**: 新创建的文件
- **修改文件**: 内容有变化的文件
- **删除文件**: 已删除的文件

### 预览差异

1. 点击同步面板中的文件
2. 查看变更详情
3. 添加的行显示绿色
4. 删除的行显示红色

### 同步操作

| 操作 | 说明 |
|------|------|
| 同步全部 | 将所有变更应用到本地 |
| 同步选中 | 只同步选中的文件 |
| 撤销 | 撤销未保存的变更 |

## 隐私与安全

### 数据存储

- 所有文件保存在本地
- AI 处理在本地浏览器完成
- 不会上传文件到云端

### 连接安全

- 远程连接使用加密通道
- 支持设置访问密码
- 可以随时断开连接

### 断开连接

点击"断开连接"按钮：

- 立即断开远程访问
- 清除连接缓存
- 需要重新扫码连接

## 常见问题

### 连接失败

1. 确认电脑和手机在同一网络
2. 检查防火墙设置
3. 尝试重新生成连接码

### 同步不工作

1. 刷新页面重试
2. 检查浏览器权限
3. 确认文件路径是否正确
