# 工作空间

工作空间是 CreatorWeave 的核心功能区域，在这里你可以管理项目、浏览文件、与 AI 协作。

## 项目管理

### 创建新项目

1. 点击顶部导航栏的项目名称
2. 在下拉菜单中选择"新建项目"
3. 选择本地文件夹
4. 等待 AI 分析项目结构

### 切换项目

1. 点击项目切换器
2. 查看最近项目列表
3. 点击任意项目切换

### 项目设置

每个项目可以独立配置：

| 设置项 | 说明 |
|-------|------|
| AI 模型 | 选择使用的 AI 模型 |
| 系统提示词 | 自定义 AI 的行为指令 |
| 工具权限 | 控制 AI 可以执行的操作 |

## 文件浏览

### 文件树

左侧边栏显示项目文件结构：

- 点击文件夹展开/折叠
- 双击文件打开预览
- 右键文件查看更多操作

### 文件预览

支持的预览格式：

- 源代码文件（语法高亮）
- Markdown 文档
- 图片文件
- JSON/YAML 配置

### 搜索文件

使用搜索功能快速定位文件：

1. 点击搜索图标或使用快捷键
2. 输入文件名或内容关键词
3. 点击结果跳转到文件

## 与 AI 协作

### 发送指令

在底部输入框中输入自然语言指令：

```
帮我修改 src/components/Button.tsx 的样式
```

### 引用项目文件

使用 `@` 符号引用文件：

```
@src/App.tsx
分析这个文件的主要功能
```

### 查看执行结果

AI 执行操作后会展示：

- 修改的文件列表
- 变更内容预览
- 执行日志和状态

## 快捷键

| 快捷键 | 功能 |
|-------|------|
| `Cmd/Ctrl + K` | 打开命令面板 |
| `Cmd/Ctrl + P` | 快速打开文件 |
| `Cmd/Ctrl + N` | 新建对话 |
