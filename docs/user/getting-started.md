# 快速入门

欢迎使用 CreatorWeave！这是一个 AI 原生的开发者工作空间，让你可以用自然语言与本地文件交互。

## 准备工作

### 环境要求

- **浏览器**: Chrome 110+, Firefox 120+, Safari 16+, Edge 110+
- **网络**: 需要访问 AI API 服务

### 首次设置

1. 打开 CreatorWeave 应用
2. 点击顶部导航栏的设置图标
3. 在"API 设置"中配置你的 AI API Key
4. 保存设置

## 开始使用

### 创建第一个项目

1. 点击顶部的项目切换器
2. 选择"新建项目"
3. 选择一个本地文件夹作为项目目录
4. AI 会自动分析项目结构

### 与 AI 对话

在底部的输入框中输入你的需求：

```
帮我创建一个用户登录组件
```

AI 会理解你的需求，执行必要的操作，并展示结果。

### 常用指令示例

| 指令类型 | 示例 |
|---------|------|
| 创建文件 | 创建一个 React 组件用于显示用户信息 |
| 解释代码 | 解释 src/utils/helpers.ts 中这段代码的作用 |
| 优化性能 | 帮我优化这个函数的性能 |
| 生成测试 | 为这个模块生成单元测试 |
| 代码审查 | 审查这个 PR 的代码改动 |

## 核心概念

### 项目

一个项目对应本地的一个文件夹。所有对话、文件操作都在项目范围内进行。

### 对话

每个对话是一个独立的会话。你可以为不同的任务创建不同的对话。

### Agent 模式

开启后，AI 会主动规划并执行一系列操作来完成复杂任务。
