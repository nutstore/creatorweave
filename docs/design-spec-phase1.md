## 2. 组件规范

### 2.1 按钮 (Button)

#### Primary Button

**视觉设计**

```
默认状态:
┌──────────────────────┐
│  [选择文件夹]         │  背景: #3B82F6
│                      │  文字: #FFFFFF
└──────────────────────┘  圆角: 8px
   宽度: 自适应           高度: 44px
   内边距: 12px 24px      字重: 500
```

**状态规范**

| 状态 | 背景色 | 文字色 | 边框 | 阴影 | 其他 |
|------|--------|--------|------|------|------|
| Default | `#3B82F6` | `#FFFFFF` | 无 | 无 | - |
| Hover | `#2563EB` | `#FFFFFF` | 无 | `0 4px 12px rgba(59, 130, 246, 0.4)` | - |
| Active | `#1D4ED8` | `#FFFFFF` | 无 | `0 2px 4px rgba(59, 130, 246, 0.4)` | transform: scale(0.98) |
| Disabled | `#93C5FD` | `#FFFFFF` | 无 | 无 | opacity: 0.6, cursor: not-allowed |
| Loading | `#3B82F6` | `#FFFFFF` | 无 | 无 | 显示 spinner, 禁用点击 |

**尺寸变体**

| 尺寸 | 高度 | 内边距 | 字体大小 | 图标大小 |
|------|------|--------|----------|----------|
| Small | 36px | 8px 16px | 14px | 16px |
| Medium (默认) | 44px | 12px 24px | 16px | 20px |
| Large | 52px | 16px 32px | 18px | 24px |

**无障碍要求**

- 焦点可见性: 2px 蓝色边框 (`#3B82F6`)
- 键盘可访问: 支持 Tab 和 Enter/Space
- ARIA 属性: `role="button"`, `aria-label`, `aria-disabled`
- 最小触摸目标: 44x44px (WCAG AAA 标准)

## 7. 动画规范

### 7.1 缓动函数 (Easing Functions)

| 缓动函数 | CSS 值 | 应用场景 |
|----------|--------|----------|
| Ease | `ease` | 默认过渡 |
| Ease In | `ease-in` | 进入动画 |
| Ease Out | `ease-out` | 退出动画 |
| Ease In Out | `ease-in-out` | 进出动画 |
| Custom Bounce | `cubic-bezier(0.68, -0.55, 0.265, 1.55)` | 弹性效果 |
| Custom Smooth | `cubic-bezier(0.4, 0, 0.2, 1)` | 平滑过渡 |

### 7.2 动画持续时间

| 动画类型 | 持续时间 | 缓动函数 |
|----------|----------|----------|
| 淡入 (Fade In) | 300ms | `ease-in-out` |
| 淡出 (Fade Out) | 200ms | `ease-in-out` |
| 滑入 (Slide In) | 400ms | `ease-out` |
| 滑出 (Slide Out) | 300ms | `ease-in` |
| 缩放 (Scale) | 200ms | `ease-out` |
| 进度条更新 | 300ms | `ease-out` |
| 数字滚动 | 1000ms | `ease-out` |
| 按钮悬停 | 200ms | `ease-out` |
| 卡片悬停 | 300ms | `ease-out` |
| 旋转 (Spinner) | 1000ms | `linear` (infinite) |
| 抖动 (Shake) | 500ms | `cubic-bezier(0.36, 0.07, 0.19, 0.97)` |

### 7.5 动画性能优化

#### GPU 加速

```css
/* 使用 transform 和 opacity */
.animated-element {
  will-change: transform, opacity;
  transform: translateZ(0);
}

/* 避免使用 left, top, width, height */
.bad-performance {
  left: 100px; /* ❌ 触发重排 */
}

.good-performance {
  transform: translateX(100px); /* ✅ GPU 加速 */
}
```

#### 减少重绘

```css
/* 使用 requestAnimationFrame */
.animated-element {
  animation: slideIn 400ms ease-out;
}

/* 避免强制同步布局 */
.no-layout-thrashing {
  /* ❌ 不要在动画中读取布局属性 */
  const height = element.offsetHeight;
  element.style.height = height + 10 + 'px';
}

.layout-friendly {
  /* ✅ 使用 transform 代替 */
  element.style.transform = 'translateY(10px)';
}
```

### 9.3 设计决策记录

#### 为什么使用系统字体?
- **性能**: 无需加载外部字体文件
- **一致性**: 用户熟悉的字体
- **本地化**: 自动适配系统语言

#### 为什么使用 4px 基础间距单位?
- **一致性**: 4px 可整除大多数屏幕尺寸
- **可预测**: 便于计算和调整
- **行业标准**: 与主流设计系统一致

#### 为什么选择蓝色作为主色调?
- **信任感**: 蓝色传达可靠和信任
- **专业性**: 适合工具类应用
- **可访问性**: 蓝色具有良好的对比度
- **行业标准**: 技术产品常用色
