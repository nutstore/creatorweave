# Sync and Remote Control

CreatorWeave supports remote session workflows through the relay server.

## Typical Use Cases

- Continue monitoring a session from mobile.
- Review progress while away from your desktop.
- Coordinate handoff across devices.

## Security Model

- Sessions are designed with encrypted transport and key exchange support.
- Local-first behavior remains the default for core file workflows.
