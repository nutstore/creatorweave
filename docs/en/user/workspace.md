# Workspace

The workspace is the main surface where you organize files, conversations, and tools.

## Main Areas

- **Sidebar**: project switching, navigation, and quick access to key sections.
- **Conversation panel**: AI chat, tool calls, and response history.
- **File and preview panels**: inspect source files, diffs, and generated artifacts.
- **Command palette**: fast keyboard-driven actions (`Ctrl/Cmd + K`).

## Best Practices

- Keep each conversation focused on one problem.
- Use project/workspace separation for different contexts.
- Review tool outputs before applying large file updates.

## Keyboard Shortcuts

- `Ctrl/Cmd + K`: open command palette
- `Ctrl/Cmd + B`: toggle sidebar
- `Shift + ?`: show shortcuts
