# Getting Started

AI Workspace is a local-first workspace for creators and engineering teams.

## Prerequisites

- A modern Chromium-based browser (recommended)
- Node.js 18+
- pnpm

## Quick Start

```bash
git clone https://github.com/nutstore/creatorweave.git
cd creatorweave
pnpm install
pnpm -C web run dev
```

Open `http://localhost:5173`.

## Core Workflow

1. Create or open a project.
2. Grant folder access so the workspace can read files.
3. Start a conversation with AI and use tools to inspect or modify files.
4. Review and export your results.
