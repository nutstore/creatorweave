# Conversations

Conversations connect your prompts, tool calls, and outputs in one timeline.

## What You Can Do

- Ask for code analysis, refactoring, and debugging.
- Let the agent run tools for file discovery and batch operations.
- Regenerate responses and compare alternatives.

## Suggested Prompt Pattern

Use this structure for better output quality:

1. **Goal**: what you want to achieve.
2. **Context**: relevant files, constraints, and runtime assumptions.
3. **Success criteria**: what “done” looks like.

## Safety Notes

- Review generated file edits before applying them.
- Keep backups or version control checkpoints for large changes.
