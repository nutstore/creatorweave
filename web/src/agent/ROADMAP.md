# Agent Tools Roadmap

## 长期特性

### 1. 文本搜索 (search_text)

**问题：** 当前 `read_directory` 只有列表功能，没有搜索功能。

**实现方案：**

| 方案 | 描述 | 优点 | 缺点 |
|------|------|------|------|
| A | 复用 execute + Python grep | 实现简单，直接用现成工具 | 需要加载 Pyodide |
| B | Pyodide 封装 ripgrep | 效率高，Linux 原生体验 | 需要维护 ripgrep WASM |
| C | 预建索引 | 搜索极快 | 需要维护索引，初始慢 |

**推荐：** 方案 A 或 B

**待定：**
- 是否需要实时搜索 (边输入边搜索)？
- 是否需要支持大项目 (1000+ 文件)？
